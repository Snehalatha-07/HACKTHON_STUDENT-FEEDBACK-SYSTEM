# UI Wireframes (low-fidelity)

Below are low-fidelity wireframe outlines and flow descriptions for key screens. These are intentionally simple and focus on layout and primary actions.

## 1) Landing / Home
- Top nav: logo | Products (Admin / Student) | Login
- Hero: Title, short description, CTA `Get Started` / `Login`
- Feature tiles: Quick statistics, Contact

## 2) Admin Dashboard
- Left nav: Dashboard, Forms, Responses, Analytics, Courses, Settings
- Main: Quick-actions (Create Form, View Responses, Seed Data), Summary cards (Total Forms, Total Responses, Response Rate)

## 3) Form Builder (Drag & Drop)
- Left pane: Question bank + Add question types
- Middle canvas: Question list with reorder handles
- Right pane: Question settings (required, scale, options)
- Top: Save / Preview / Publish

## 4) Analytics
- Top: Date / Form / Course filters + Palette selector
- Left: Overview cards (Total Responses, Avg Rating, Active Forms)
- Center: Responses by Course (Bar chart) + Download buttons
- Bottom: Form list with quick sparkline + `View Details`

## 5) Admin Responses (Table)
- Filters row: Course, Form, Search, Include student IDs, Reset
- Table: ID | Form | Course | Student | Rating | Comment | Submitted
- Pagination controls and Export CSV button

## 6) Student Dashboard
- Pending forms, Completed forms, Quick Feedback
- Notifications for new forms / reminders

## 7) Student Feedback Form
- Progress bar, questions as single page or multi-step
- Required indicators, Save Draft, Submit
